# Week7 Full Stack Notes Application using Node and Express

## Description
A simple web application for creating, viewing, editing, and deleting notes.
Used p4_interactive_todo_list as basis, updated html and javascript, used Node.js and Express to convert it to a full stack notes applicaiton.

## Installation
1. Clone the code locally
2. Install dependencies
    bash
    npm init -y
    npm install express
3. Start the server 
    bash
    node server.js
4. Access the application
    Open your browser and navigate to: http://localhost:3001

## Usage
1. Adding a Note:
    1) Select a date (must be today or future)
    2) Enter note details in the text field
    3) Click "Add Note" to save
2. Editing a Note:
    1) Click the "Edit" button next to any note
    2) Enter the new note details in the prompt
    3) The note will be updated immediately
3. Deleting a Note:
    1) Click the "Delete" button next to any note
    2) The note will be removed immediately
